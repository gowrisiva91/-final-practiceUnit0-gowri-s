# Smart Aquarium Feeder App 🐠

## 📌 Description

The Smart Aquarium Feeder App is a simple JavaScript-based project designed to help aquarium owners manage their fish feeding schedule. This app allows users to track feeding times, display reminders, and organize information about different fish in one place.

## 🎯 Purpose

This project was created to practice foundational JavaScript concepts from Unit 0, including variables, strings, arrays, loops, and objects.

## ⚙️ Features

* Store and display fish feeding information
* Generate reminder messages for feeding
* Check feeding time using conditional logic
* Manage a list of fish using arrays
* Loop through fish data using `forEach()`
* Store fish details using objects

## 🧠 Skills Used

This project demonstrates the following JavaScript skills:

* **Values, Data Types, and Operations** – storing fish name and food quantity
* **Stringing Characters Together** – creating reminder messages
* **Control Structures and Logic** – checking feeding time with `if/else`
* **Building Arrays** – storing multiple fish names
* **Using Arrays** – looping through the fish list with `forEach()`
* **Creating and Using Objects** – storing fish details like name and feeding time
